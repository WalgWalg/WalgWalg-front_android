# WalgWalg

---
산책로 추천 어플리케이션

- 기능
    - 걸음수,km,칼로리 기록
    - 로그인, 회원가입
    - 오늘날씨
    - 위치기반 산책코스 추천받기(경기도)
    - 내 산책로 저장 및 추천해주기
    → 검색
    → 게시판 형식
    → 좋아요(추천순으로 보여주기 가능)
    → 스크랩
    - 보기쉽게 달력형식으로 산책기록
        
        → km, 위치, 지도(기록)
        
    - 마이페이지 (좋아요, 자기산책로)
    
- 역할
    - 준하
        - 기능
            - 게시판 CRUD
            - 좋아요 기능
            - 마이페이지(요약 정보, 산책 기록 조회)
            - 게시판 검색 기능
        - 사용기술
            - ConstraintLayout과 GuideLine을 활용하여 핸드폰 해상도에 따른 화면 잘림 문제 최소화
            - Http 통신을 위한 Retrofit 사용
            - 직관적인 UI를 위한 Material 활용
            
- 아쉬운점
    - 준하
        - 서버로부터 값을 받아올 때 소요되는 시간동안 띄울 로딩화면 구현
        - ConstraintLayout과 GuideLine에 대한 자세한 이해
        - MVVM 패턴에 대한 자세한 이해
